# RESOLVE cli

The primary user interface to RESOLVE; a collection of subcommands where
`resolve <subcommand> [args...]` runs `resolve-<subcommand> [args...]`.

**Full documentation:** <https://riversideresearch.github.io/resolve/components/resolve-cli/resolve-cli/>

The documentation site is the canonical source for setup and usage, including
the per-subcommand pages for
[SBOM](https://riversideresearch.github.io/resolve/components/resolve-cli/sbom/),
[crash analyzer](https://riversideresearch.github.io/resolve/components/resolve-cli/crash-analyzer/),
and [input synthesis](https://riversideresearch.github.io/resolve/components/resolve-cli/input-synthesis/).
