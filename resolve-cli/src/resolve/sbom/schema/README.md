# Schema Classes

This directory contains python representations of the NIST API schema and RESOLVE vulnerability description schema.

## Generating

This project uses pydatnic for its API representations, allowing for automatic validation of types and easy (de)serialization to/from JSON. 

### From JSON Schema

The API schema were generated using the [datamodel-code-generator](https://pypi.org/project/datamodel-code-generator) tool.

An example workflow for this project being:
```bash
curl -O https://csrc.nist.gov/schema/nvd/api/2.0/cve_api_json_2.0.schema
datamodel-codegen --input cve_api_json_2.0.schema --input-file-type jsonschema --output cve_api
```

The schema definitions used to generate files in this directory are available in `../schema_defs/`. 

### By hand

The RESOLVE `vulnerabilities.json` format is not defined by an official JSON schema, as was generated by hand, using examples and existing parser code as a reference.
